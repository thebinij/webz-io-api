# webz-io-api

## Installation

Install the dependencies:

```bash
yarn install
```

Set the environment variables:

```bash
cp .env.example .env

# open .env and modify the environment variables (if needed)
```

## Commands

Running locally:

```bash
yarn dev
```

Kill locally:

```bash
yarn kill
```

Running in production:

```bash
yarn start:dev
```

Testing:

```bash
# run all tests
yarn test
```

Build:

```bash
yarn build
```
